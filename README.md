# anagram-classes

Sort words into anagram classes with identical letter-sets, e.g.

  * kitchener   
  * tiffy     
  * rethicken     
  * hands    
  * thickener    
  * fifty    

Would be organised:

  * ['kitchener', 'rethicken', 'thickener']   
  * ['fifty', 'tiffy']   
  * ['hands']
